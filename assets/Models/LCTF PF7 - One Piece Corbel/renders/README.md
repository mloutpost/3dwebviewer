# Rendered Images for LCTF Barn

This directory contains rendered images that will be displayed in the model details gallery.

## How to Add Images

1. Place your rendered images (PNG, JPG, JPEG, WebP) in this `renders/` directory
2. Name them descriptively, e.g.:
   - `exterior-front.png`
   - `interior-living-space.jpg`
   - `floor-plan.png`
   - `elevation-north.png`

3. The images will automatically appear in the gallery when you visit the model details page

## Supported Formats
- PNG
- JPG/JPEG  
- WebP

## Image Recommendations
- **Resolution**: 1200x800px or higher for best quality
- **Aspect Ratio**: 3:2 or 16:10 works well
- **File Size**: Keep under 2MB per image for fast loading
- **Naming**: Use descriptive names that indicate what the image shows

## Example Structure
```
renders/
├── exterior-front.png
├── interior-main-floor.jpg
├── floor-plan.png
├── elevation-south.png
└── README.md
```
